This data structures assignment uses different data structures like graphs and queues
to allocate ressources to cities from warehouses in an emergency situation. 
It does the allocations to cities that have the most priority and also does redistribution
of the ressources within the warehouses.

To run the project:
Go to TP2/src. There use the makefile to run the commands. make clean, make compile
make run. The results from TestCase1&2.txt should be output in the json files.
